# Personal Portfolio

A responsive personal portfolio website built to showcase my projects, skills, and experience as a Full Stack Developer. 

## Technologies Used
* React.js
* JavaScript
* HTML5 & CSS3

## How to Run Locally
1. Clone the repository: `git clone https://github.com/Mosees33/Portfolio-React.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
